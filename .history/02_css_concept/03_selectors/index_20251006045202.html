CSS Selectors

What are CSS Selectors?

CSS selectors are patterns used to select and style HTML elements. They allow you to apply styles to specific elements based on their attributes, classes, IDs, or relationships with other elements.

Types of CSS Selectors

  • Element Selector: Selects all instances of a specific HTML element. Example: p { color: blue; }
  • Class Selector: Selects all elements with a specific class attribute. Example: .my-class { font-size: 16px; }
  • ID Selector: Selects a single element with a specific ID attribute. Example: #my-id { background-color: yellow; }
  • Attribute Selector: Selects elements based on the presence or value of an attribute. Example: [type="text"] { border: 1px solid black; }
  • Pseudo-class Selector: Selects elements based on their state or position. Example: a:hover { text-decoration: underline; }
  • Descendant Selector: Selects elements that are descendants of a specified element. Example: div p { margin-bottom: 10px; }

Combining Selectors

You can combine multiple selectors to target more specific elements. For example:

  • div.my-class { color: red; } - selects all <div> elements with the class "my-class".
  • #my-id p { font-weight: bold; } - selects all <p> elements inside the element with ID "my-id".
