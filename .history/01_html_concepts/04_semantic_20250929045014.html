Understanding HTML Semantics

HTML semantics refers to the use of HTML tags that convey meaning about the content they enclose. Using semantic tags improves accessibility, SEO, and code readability.

Common Semantic Tags

  • <header>: Represents introductory content or a set of navigational links.
  • <nav>: Defines a set of navigation links.
  • <main>: Specifies the main content of a document.
  • <article>: Represents a self-contained piece of content.
  • <section>: Defines a section in a document.
  • <footer>: Represents the footer for its nearest sectioning content or sectioning root element.
